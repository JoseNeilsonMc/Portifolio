HTML5 é a quinta versão da Linguagem de Marcação de Hipertexto (HTML),
utilizada na construção e estruturação de páginas web. Introduziu novos elementos e
atributos, oferecendo maior suporte para multimídia, gráficos e aplicativos interativos.

CSS3, ou Cascading Style Sheets 3, é a terceira versão das folhas de estilo em cascata,
responsáveis por controlar a apresentação visual de documentos HTML. Oferece recursos avançados,
como transições, animações e sombras, permitindo uma estilização mais sofisticada e responsiva.

O Bootstrap é um framework de design front-end popular para criar sites e aplicativos responsivos.
Desenvolvido pelo Twitter, ele oferece componentes pré-construídos e de uma forma flexível para facilitar
o desenvolvimento. Sua simplicidade, personalização e compatibilidade com navegadores tornam-no uma escolha
amplamente exigida por desenvolvedores para criar interfaces web modernas e consistentes em diferentes dispositivos.

JavaScript é uma linguagem de programação de alto nível que possibilita a criação de conteúdo
dinâmico e interativo em páginas web. Executado no navegador do usuário, o JavaScript pode manipular
o conteúdo da página, responder a eventos e facilitar a comunicação com servidores.

React é uma biblioteca de JavaScript desenvolvida pelo Facebook, utilizada para a construção
de interfaces de usuário (UI) interativas e eficientes. Baseado em componentes reutilizáveis,
React permite a criação de aplicativos web de uma maneira mais modular e fácil de manter.

Git é um sistema de controle de versão distribuído amplamente utilizado no desenvolvimento
de software. Permite que equipes de desenvolvimento colaborem de maneira eficiente, mantendo um
histórico detalhado das alterações no código-fonte. Git facilita o rastreamento de versões, fusão de alterações
e colaboração entre desenvolvedores.
